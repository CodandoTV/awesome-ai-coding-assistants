# Studies

This folder contains practical studies and experiments related to AI coding assistants.

Each study includes raw data, visual summaries, and a short README explaining the goal, methodology, and main findings.

## Available Studies

### Caveman Token Usage Benchmark

A practical benchmark about [Caveman](https://github.com/JuliusBrussee/caveman) usage inside Claude Code.

The study compares token/context usage across:

- Random questions vs code questions
- Portuguese vs English prompts
- Caveman disabled vs Caveman full mode
- Caveman full vs ultra mode
