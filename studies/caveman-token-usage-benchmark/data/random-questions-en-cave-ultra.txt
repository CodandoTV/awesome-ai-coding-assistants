РЮ» How does sleep work? Why do we need to sleep?                                 

РЈ║ Brain cycles through NREM/REM stages. Each ~90min.

  Why needed:
  - Glymphatic system flush toxic waste (amyloid-╬▓, tau) Рєњ skip sleep Рєњ buildup
  Рєњ Alzheimer's risk
  - Memory consolidation: hippocampus Рєњ cortex transfer during slow-wave
  - Synaptic pruning: weaken weak connections, strengthen strong Рєњ learning
  - Hormone reset: growth hormone peak during deep sleep, cortisol drop
  - Immune repair: cytokine production up

  Skip sleep Рєњ cascade:
  no sleep Рєњ cortisol spike Рєњ inflammation Рєњ impaired glucose metabolism Рєњ 
  cognitive decline

  Why can't evolution remove it: Every animal sleeps. Cost (vulnerable,
  immobile) outweighed by benefit Рєњ deeply conserved. No workaround found in
  500M years of evolution.

  REM = emotional memory + creativity. NREM = physical repair + declarative
  memory. Need both.

РЮ» How does human memory work? Why do we forget things?                          

РЈ║ Encoding Рєњ Storage Рєњ Retrieval

  sensory input Рєњ working mem (7┬▒2 items, ~20s)
                      РєЊ (rehearsal/emotion/attention)
                long-term mem (hippocampus tags it)
                      РєЊ (sleep consolidation)
                cortex (permanent storage)

  Types:
  - Declarative: facts + episodes (hippocampus-dependent)
  - Procedural: skills (basal ganglia, cerebellum)
  - Emotional: fear/reward (amygdala amplifies encoding)

  Why forget:

  1. Decay Рђћ unused synaptic connections weaken (use it or lose it)
  2. Interference Рђћ similar memories overwrite each other
  (proactive/retroactive)
  3. Retrieval failure Рђћ memory exists, path broken (tip-of-tongue = this)
  4. Active suppression Рђћ prefrontal cortex inhibits unwanted memories
  5. Encoding failure Рђћ never stored properly (phone in hand, no memory of
  putting it down)

  Forgetting curve (Ebbinghaus): 50% lost in 1hr without review Рєњ spaced 
  repetition exploits this

  Key insight: Forgetting = feature not bug. Brain prunes low-signal data Рєњ
  reduces noise Рєњ better pattern recognition. Perfect memory Рєњ worse decisions
  (see: hyperthymesia cases).

  Emotion + novelty + repetition Рєњ strong encoding Рєњ harder to forget.

РЮ» Why is the sky blue during the day and orange/red at sunset?                  

РЈ║ Rayleigh scattering.

  Sunlight = all wavelengths. Atmosphere = NРѓѓ/OРѓѓ molecules scatter short
  wavelengths >> long.

  scatter РѕЮ 1/╬╗РЂ┤

  Blue (╬╗~450nm) scattered ~5.5x more than red (╬╗~700nm).

  Day: Sun overhead Рєњ short path through atmo Рєњ blue scattered everywhere Рєњ sky
  looks blue. Eyes see scattered blue from all directions.

  Sunset: Sun at horizon Рєњ light travels ~40x longer path through atmo Рєњ
  blue/green fully scattered away before reaching eye Рєњ only red/orange survives
   Рєњ direct light is warm.

  Bonus: Mars sky opposite Рђћ dust particles larger Рєњ scatter red, not blue Рєњ
  blue near sun at sunset, pink/red elsewhere.

РЮ» How does a vaccine work? Why does it protect against diseases?                

РЈ║ Immune system trains on fake threat.

  Normal infection path:
  pathogen enters Рєњ immune system detects antigens
  Рєњ B cells produce antibodies (days-weeks)
  Рєњ T cells kill infected cells
  Рєњ memory cells persist after clearance
  First exposure = slow Рєњ often sick before response wins.

  Vaccine:
  Deliver antigen (dead/weakened pathogen, protein fragment, or mRNA
  instructions) without actual infection risk.

  vaccine Рєњ immune response triggered
  Рєњ memory B + T cells created
  Рєњ real pathogen arrives Рєњ memory cells recognize instantly
  Рєњ antibodies flood Рєњ pathogen neutralized before symptoms

  mRNA vaccines (COVID): Don't inject protein Рђћ inject instructions. Cells make
  spike protein themselves Рєњ immune system trains on it Рєњ mRNA degrades in days,
   never touches DNA.

  Why protection fades:
  Memory cell populations decline over time. Boosters re-stimulate. Some
  vaccines (measles) Рєњ lifelong. Others (flu) Рєњ virus mutates Рєњ annual update
  needed.

  Herd immunity: When enough % immune Рєњ pathogen can't find hosts Рєњ transmission
   chain breaks Рєњ protects unvaccinated too. Threshold varies by RРѓђ (measles
  ~95%, COVID ~70-85%).

  Core insight: immune system = pattern matcher. Vaccine = safe training data.
